# Pendle – Track What Matters Most

Pendle is a one-page website showcasing the **Pendle app**, designed to help you track durations, milestones, and personal metrics in a simple, visual way. No complex dashboards, no mental overload—just clarity at a glance.

![Pendle App Interface](path-to-screenshot.png)

## About

Many assume people can easily process text, numbers, and time, but that’s not true for everyone. Visual thinkers, people with dyslexia, or anyone who struggles with remembering numbers can find it difficult to answer questions like:

- “How long has it been since…?”  
- “When did I last…?”

Pendle solves this by turning important moments into **intuitive visual timelines and signals**, so you instantly understand what matters most.

## Features

- Visual duration tracking  
- Milestones and personal metrics  
- Easy-to-read timelines  
- Simple, distraction-free interface
